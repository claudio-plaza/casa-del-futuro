Biblioteca Virtual - POO
-------------------------
Arquitectura y archivos:
- author.js  -> clase Author
- book.js    -> clase Book (encapsulación y métodos)
- library.js -> gestión de colección (agregar, pedir, devolver, eliminar)
- main.js    -> UI y orquestación
- styles.css -> diseño moderno

Cómo abrir:
1. Abrir 'index.html' en el navegador (doble clic).
2. Agregar libros con título, autor, año y copias.
3. Usar 'Pedir' y 'Devolver' para simular préstamos.
4. Guardar manualmente con 'Guardar Biblioteca' o se guarda automáticamente al agregar.

Conceptos aplicados:
- Clases, encapsulación (#private), métodos estáticos
- Herencia y composición (Author dentro de Book)
- localStorage para persistencia (se reconstruyen instancias al cargar)
